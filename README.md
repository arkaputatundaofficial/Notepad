# Notepad

This project brings you the features I've always wanted in a Notepad but wasn't fortunate enough to find anything similar so I built it anyways. Spotlight feature include:
- **"Folderization" of Notes**: Ability to arrange and group notes into "folders".

Apart from this, the app contains all the basics you'd expect. To list a few:
- Complete CRUD operation across notes
- Searching of Notes
- Flat design
